Windows10安装redis

2022-11-27 约 302 字 预计阅读 1 分钟

文章目录

    • 下载Redis
    • 安装Redis

下载Redis

1、打开https://github.com/MSOpenTech/redis/releases，选择版本下载msi安装包，编写博客时最新版本为3.2.100。

安装Redis

1、打开msi安装包，点击next

2、同意协议，点击next

3、点击Change..更改安装目录，点击复选框选择是否将应用添加到环境变量（建议勾选），点击next。

4、选择Redis的服务端口，并将该端口从Windows从防火墙中添加例外，默认的选项就可以。点击next。

5、设定Redis存储容量限制。不需要进行限制，点击next。

6、设置已完成点击install进行安装。

7、安装完成，点击finish结束安装。
